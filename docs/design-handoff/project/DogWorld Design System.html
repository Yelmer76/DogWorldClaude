DogWorld · Design system reference

The kit for the kennel.

One source of truth — the website, the back-office, the buyer portal. Quiet, disciplined, warm. Built for breeders with a phone in one hand and a puppy in the other.

Anatomy
  • Top bar — 56px. Title centered; left slot for back / menu; right slot for one contextual icon. No more.
  • Bottom tabs — 64px including safe-area. Five fixed destinations; active state uses sage green, never navy. Navy is for actions, not navigation.
  • Hit target — 44px minimum. Tabs are 64×64 effective.
  • Notifications dot — navy. Only on Inbox tab and bell icon.

Toasts announce a successful action and disappear. Dialogs only appear for destructive or irreversible actions — never to nag.

Notes
Sheet height is content-driven. Up to 60% viewport, then it scrolls. Never push the sheet over the top safe-area; the user should always see context above.
  • Dim background is rgba(26,26,26,0.32).
  • Grabber: 36×4, neutral-200. Always present on draggable sheets.
  • Destructive items live last, separated visually by colour only.
  • Tapping outside or pressing the grabber dismisses.
15 · Dog-specific

Pedigree node card

Used in the genealogy tree. The focal dog sits on the left — ancestors flow rightward back through time, matching how NKK, VDH and KC print pedigrees read. Sires carry a dusty-blue front bar; dams a dusty-pink one. Cards stay white — the sex hint reads at a glance without competing with the rest of the UI. Single tap on any ancestor opens the peek below; from there, “Make focal →” walks the tree.
