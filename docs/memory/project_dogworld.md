DogWorld (working name) is a SaaS being built for dog kennel owners / breeders. Working dir `D:\Antigravity\DogWorldClaude`.

**What it does (planned):**
- Mobile-first website builder for kennels — replaces the typical hand-built WordPress/Wix kennel site
- Manages all the standard kennel content: living dogs (breeding stock, "studs" and "dams"), sold dogs, in-memoriam ("graveyard" / Rainbow Bridge), litters/puppies for sale, show titles & competition results, veterinary / health-test records, pedigrees
- Migration tooling: scrape an existing kennel's site and import into the new SaaS
- Configurable public-facing site that the kennel can theme

**Why:** existing kennel sites are largely outdated, hand-built, not mobile-friendly. Existing kennel software (ZooEasy, Breedera, etc.) tends to be back-office / desktop-era. Gap exists for a modern, mobile-first, owner-friendly product.

**How to apply:**
- User is the founder/builder, likely Norwegian (email olebakre@gmail.com) — Norwegian Kennel Club (NKK) and FCI conventions are especially relevant alongside AKC/UKC
- When suggesting features or schemas, default to mobile-first and assume non-technical end users (kennel owners, not developers)
- Migration story is a first-class concern, not an afterthought — the wedge is "we'll move you off your old site for free"
- Project is at the research/scoping phase as of 2026-05-10 — no code committed yet
